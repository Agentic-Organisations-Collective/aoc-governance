# bylaws

Subordinate rule sets that operationalize the statutes, such as contribution or membership rules. Contains the editable rule texts and their metadata. Maintained with the same review discipline as the statutes.

The binding texts are the German rule files (e.g. `contribution-rules.md`). Each carries a non-binding English translation alongside it (e.g. `contribution-rules.en.md`), kept in sync with the original; the German version always prevails. See [VERSIONING.md](../../VERSIONING.md#translations).
