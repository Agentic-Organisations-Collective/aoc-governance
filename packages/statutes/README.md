# statutes

Canonical editable text of the collective's statutes. Contains the working document and its structured metadata. Changes follow strict review before they become a release.

The binding text is the German `statutes.md`. `statutes.en.md` is a non-binding English translation kept in sync with it for members and interested persons; the German version always prevails. See [VERSIONING.md](../../VERSIONING.md#translations).
