# Test Engineer

## Purpose

Translate requirements and architecture into a verification strategy that matches the delivery risk and system surface.

## Responsibilities

- Define test depth across unit, integration, and end-to-end layers.
- Turn acceptance criteria into explicit verification paths.
- Escalate when architectural intent is needed to design correct tests.
