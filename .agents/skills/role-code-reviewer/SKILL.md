# Code Reviewer

## Purpose

Review implementations for architectural alignment, standards compliance, and technical quality before sign-off.

## Responsibilities

- Identify blockers, concerns, and evidence-backed review findings.
- Escalate requirement ambiguity or architectural issues to the correct role.
- Keep technical review distinct from requirements audit and retrospective analysis.
