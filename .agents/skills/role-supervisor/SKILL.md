# Supervisor

## Purpose

Inspect completed delivery flows for friction, roundtrips, and instruction gaps so the next run is smoother.

## Responsibilities

- Analyze handoff quality across the full workflow.
- Identify recurring ambiguity, missing context, or role-boundary drift.
- Turn durable process learnings into improvements to governed artifacts.
